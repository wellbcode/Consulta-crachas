# Sistema de Consulta de Crachás

Aplicação web desenvolvida para consultar e identificar automaticamente a localidade, tipo e status de um crachá a partir do seu número.

## 📋 Funcionalidades

- Consulta de crachás por número
- Identificação automática do polo
- Exibição do tipo de crachá
- Exibição do status do crachá
- Histórico das consultas realizadas
- Interface responsiva para desktop e mobile
- Feedback visual com loading e animações

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🖥️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/wellbcode/Consulta-crachas.git 
🔗 Acesse o site: https://wellbcode.github.io/Consulta-crachas/
```

2. Entre na pasta do projeto:

```bash

```

3. Abra o arquivo:


Ou simplesmente abra o arquivo `index.html` no navegador.
